This is one of my all-time favorite hacks. It's a quick and easy way to format XML documents using VIM. You will need to install the utility library 'libxml2-utils' and make sure that the 'xmllint' command is in your path.

Note: All commands are assumed to be performed as 'root' unless stated otherwise. If you choose to perform this is another user, you will need to 'sudo' most of these commands.

  1. Install libxml2-utils:
    Debian/Ubuntu: apt-get install libxml2-utils
    RHEL/Fedora: yum -y install libxml2-utils
    RPM: rpm -i path_to_rpm.rpm

  2. Add the xmllint command to /usr/bin: If you installed the libxml2-utils library through any of the tools above, this should already be done for you.

  3. Modify the .vimrc file in your home directory (should not be done as root) Add the following to your .vimrc file:

    :syntax enable
function! DoFormatXml()
% !xmllint % --format
endfunction
command!
FormatXml call DoFormatXml()

  4. Try it out: Next time you are editing an XML file in VIM, just type the following command: :FormatXml
